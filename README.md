Dưới đây là các chủ đề chính mà mình đã hoàn thành:

3.  **Vòng lặp:** Tối ưu hóa tác vụ với `for`, `while`, `until`.
4.  **Nâng cao:** Viết hàm (Functions), xử lý mảng (Arrays) và quản lý tiến trình hệ thống.

## 🚀 Cách chạy Script
Để thực thi các bài tập trong kho lưu trữ này, bạn hãy làm theo các bước sau:

1.  **Clone dự án về máy:**
    ```bash
    git clone [https://github.com/lethanhdanh/ten-repo-cua-ban.git](https://github.com/lethanhdanh/ten-repo-cua-ban.git)
    cd ten-repo-cua-ban
    ```
2.  **Cấp quyền thực thi cho file:**
    ```bash
    chmod +x ten_bai_tap.sh
    ```
3.  **Chạy script:**
    ```bash
    ./ten_bai_tap.sh
    ```
